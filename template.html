Vision For The Website

Main Aspects Neccessary For The Design

The main vison for the website is accessibility as we want users to have an easy time accessing the website. Due to this, the color scheme should be pleasing to the eye and enjoyable for the user's experience, users should have an easy time page surfing, and instructions on how to interact with the website should be clear and easy to follow. With all these aspects in mind, other factors can be given more liberties to the freedom of the developer desgining in, yet it should still follow the styling and scheme presented in this reference sheet.
